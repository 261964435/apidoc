请求

数据类型: json

status:204注销成功

post /login

用户登录

请求

数据类型: json

参数:
  • username string 用户名
  • password string 密码
示例:
{
     "username": "admin",
     "password": "123456"
 }

响应-SUCCESS

status:201成功登录

参数:
  • expries int 过期的时间，单位秒
  • token string 存储 token

响应-ERROR

status:401账号密码验证错误
